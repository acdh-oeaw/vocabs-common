SSH Vocabulary Commons

This is the SSH Vocabulary Commons SKOSMOS instance. It is used to publish vocabularies that are useful for the broad Social Sciences and Humanities (SSH) that are in need of publication and specially results from SSH collaboration project such as the SSHOC project.

This SKOSMOS instance is part of a federative collaboration of a number of other SSH vocabulary publication platform instances and allows browsing and searching of externally hosted vocabularies.

Currently the Vocabulary Commons SKOSMOS instance is hosted and operated by OeAW ACDH-CH with (initial) support from the SSHOC project.

Browse page of the Vocabulary server: https://vocabs.sshopencloud.eu/browse/
